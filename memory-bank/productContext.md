# Product Context

## Target Audience

Cursor IDE users and developers who want analytics and searchability over their AI-assisted coding sessions.

## Use Cases

- Query and search past Cursor agent sessions
- Visualize development patterns and tool usage
- Semantic search across historical conversations

## Key Benefits

- Persistent, queryable record of AI-assisted development work
- Insight into tool usage patterns and session trends
- Full-text and semantic search over past interactions

## Success Criteria

- Cursor agent transcript data is reliably captured and queryable
- Dashboard provides useful visualizations of development activity
- Semantic search returns relevant past sessions

## Key Constraints

- Greenfield project — no existing code or users yet
